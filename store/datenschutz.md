## 2. Grundprinzip: Alle Daten bleiben auf Ihrem Gerät

Mikrobiom Counter speichert sämtliche Daten ausschließlich lokal auf Ihrem Gerät. Es gibt keinen Server, kein Backend, keine Cloud-Anbindung und keine Benutzerkonten. Ihre Daten verlassen Ihr Gerät zu keinem Zeitpunkt.

## 3. Welche Daten werden verarbeitet?

Die App speichert folgende Daten lokal in einer Datenbank auf Ihrem Gerät (IndexedDB):

- **Pflanzeneinträge:** Welche Pflanzenarten Sie pro Woche erfasst haben (Pflanzen-ID, Kalenderwoche, Zeitstempel)
- **Einstellungen:** Ihre persönlichen App-Einstellungen

Diese Daten werden **nicht** an uns oder Dritte übermittelt. Es erfolgt **keine** Auswertung, **kein** Tracking und **keine** Analyse durch uns.

## 4. Spracherkennung

Die App bietet eine optionale Spracheingabe über die Web Speech API des Browsers. Die Spracherkennung wird vom Betriebssystem bzw. dem zugrunde liegenden Browser (Google Chrome) bereitgestellt.

**Wichtiger Hinweis:** Je nach Gerät und Konfiguration kann die Spracherkennung Audiodaten zur Verarbeitung an Google-Server senden. Dies liegt außerhalb unserer Kontrolle und unterliegt den Datenschutzbestimmungen von Google. Die Nutzung der Spracheingabe ist vollständig optional — Sie können alle Pflanzen auch manuell eingeben.

Weitere Informationen finden Sie in der [Datenschutzerklärung von Google](https://policies.google.com/privacy).

## 5. Technische Umsetzung

Die App wird als Trusted Web Activity (TWA) über Google Chrome ausgeführt. Es gelten daher zusätzlich die Datenschutzbestimmungen von Google Chrome. Die App selbst setzt **keine** Cookies, verwendet **keine** Analyse-Tools und bindet **keine** externen Dienste ein.

## 6. Keine Datenübermittlung an Dritte

Es findet keine Übermittlung personenbezogener Daten an Dritte statt. Ausnahmen bestehen nur im Rahmen der unter Punkt 4 beschriebenen Spracherkennung, sofern Sie diese Funktion nutzen.

## 7. Datenlöschung

Da alle Daten ausschließlich auf Ihrem Gerät gespeichert sind, haben Sie jederzeit die volle Kontrolle:

- **App-Daten löschen:** Über die Android-Einstellungen (Apps → Mikrobiom Counter → Speicher → Daten löschen)
- **App deinstallieren:** Entfernt alle gespeicherten Daten vollständig
- **Export/Import:** Die App bietet eine Funktion zum Exportieren und Importieren Ihrer Daten als Datei
